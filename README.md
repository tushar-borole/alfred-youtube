# alfred-youtube

> Search YouTube


## Install

```
$ npm i --g alfred-youtube
```

*Requires [Node.js](https://nodejs.org) 4+ and the Alfred [Powerpack](https://www.alfredapp.com/powerpack/).*


## Usage

In Alfred, type `yt`, <kbd>Enter</kbd>, and your query.
